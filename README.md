# 🌤️ PakSky - Premium Java Swing Weather Ecosystem

**PakSky** is a high-performance, desktop-based weather application built with **Java Swing**. It provides a sleek, modern interface for checking atmospheric conditions across a comprehensive database of over 80 major cities in Pakistan. Designed with a premium aesthetic, it offers a smooth and intuitive user experience for desktop users.


---


## 🌟 Key Pillars of the Project

### 🖥️ 1. Modern Java Swing GUI
A high-end desktop experience featuring:
- **Steel Blue Aesthetic:** A curated color palette using `Steel Blue` and `Dodger Blue` for a professional look.
- **Responsive Layout:** Centered window positioning and optimized component bounds for clarity.
- **Dynamic Feedback:** Real-time updates for temperature and weather conditions (currently simulation-based).

### 🇵🇰 2. Comprehensive Pakistan City Database
- **Urban Coverage:** Pre-configured validation for 80+ cities including Karachi, Lahore, Islamabad, Bahawalpur, and more.
- **Smart Validation:** Case-insensitive search logic that ensures users find their city regardless of capitalization.

### ⚙️ 3. Robust Java Architecture
- **Event-Driven Programming:** Efficient handling of user actions via `ActionListener`.
- **Thread Safety:** GUI initialization on the Event Dispatch Thread (EDT) using `SwingUtilities.invokeLater`.

---

## 🚀 Tech Stack

- **Language:** Java 8+
- **Framework:** Java Swing (GUI)
- **Library:** Java AWT (Layout & Design)
- **Validation:** Java Collections Framework (ArrayList/Arrays)

## 🛠️ Installation & Setup

### 1. Prerequisites
- Ensure you have **JDK (Java Development Kit)** installed.

### 2. Running the App
1. Clone the repository:
   ```bash
   git clone https://github.com/NoorMustafa4556/PakSky-Swing-Weather-App-Java.git
   ```
2. Navigate to the folder:
   ```bash
   cd Weather-Java-Project
   ```
3. Compile the Java file:
   ```bash
   javac MyWeatherAppGUI.java
   ```
4. Run the application:
   ```bash
   java MyWeatherAppGUI
   ```
